LevelsExtractor.py - Утилита для переноса уровней из Exel таблицы в игровой формат
Применение:
1. В командной строке перейти в папку Tools
(ввести cd и перетащить в консоль папку Tools, далее нажать Enter)
2. Ввести команду python LevelsExtractor.py далее путь к талице с балансом
(можно перетащить как в пункте #1)
3. Смотрим что вывелось в консоль, если success, то уровни успешно созданы
